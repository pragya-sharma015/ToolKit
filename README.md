# Tool Kit

Web Utility Lab is a browser-based application built using ASP.NET (.NET 6) that provides a collection of useful tools and mini applications in one place. 

The project acts as a utility hub where users can access different utilities such as calculators, generators, and small interactive tools through a simple web interface.

## Features

- Age Calculator – Calculate age based on date of birth
- Unit Converter – Convert between different measurement units
- Password Generator – Generate strong random passwords
- Random Quote Generator – Display random motivational quotes
- Random Name Picker – Pick a random name from a list
- Number Guessing Game – Interactive number guessing game
- To-Do List – Manage daily tasks
- Weather Checker – Check weather information for a location

## Technologies Used

- ASP.NET (.NET 6)
- C#
- HTML
- CSS
- JavaScript

## Purpose of the Project

This project was developed to demonstrate the implementation of multiple web utilities within a single ASP.NET application. It showcases practical web development concepts including UI interaction, application logic, and responsive design.

## How to Run the Project

1. Clone the repository
2. Open the project in Visual Studio
3. Run the application
4. The project will open in your browser
